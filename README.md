# Asteroids

Asteroids is a small Python arcade game built with Pygame and managed with uv.

## Overview

This project uses uv for dependency management and execution. It keeps the Python environment isolated and includes Pygame as the game library.

## Requirements

- Python 3.13+
- uv

## Setup

Create a virtual environment and install dependencies:

```bash
uv venv
source .venv/bin/activate
uv sync
```

## Run

Start the program with:

```bash
uv run main.py
```
